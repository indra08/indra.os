Overview

About the project

ForYou is a comprehensive Loyalty App from SinarmasLand used as a promotional media platform, point acquisition system, voucher marketplace, and gamification engine — all in one unified experience across mobile and web.

As Tech Lead, I led this project end-to-end — determining the entire application infrastructure, selecting the right technologies across Mobile, Frontend, Backend, and Admin Panel stacks, while continuously validating SinarmasLand's evolving business requirements throughout the development lifecycle.

My Role

Led technical architecture decisions across 4 platforms

Selected and validated the full technology stack

Managed infrastructure (CI/CD, deployment, monitoring)

Led Agile sprints and cross-team coordination

Validated business requirements with SinarmasLand stakeholders

The Challenge

Building a scalable loyalty platform that integrates promotional campaigns, point systems, voucher transactions, and gamification modules into a seamless experience across mobile and web platforms. The system needed to handle peak transaction loads during promotional events while maintaining sub-second response times.

The Solution

I architected the full technology stack — Flutter for cross-platform mobile (iOS & Android), React JS for the customer-facing web app (foryoubysml.com), Laravel for backend API services, MySQL for reliable transactional data, and Redis for caching and session management. A dedicated Admin Panel handles management and reporting. The modular architecture ensures each component scales independently.
