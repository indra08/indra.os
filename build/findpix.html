Overview

About the project

FindPix adalah platform dokumentasi event yang menyatukan tiga pihak: event organizer, photographer, dan peserta event. Peserta dapat mencari dokumentasi foto mereka berdasarkan wajah atau bib number dari event yang telah mereka ikuti.

Platform ini menggabungkan infrastruktur berkinerja tinggi dengan kecepatan aplikasi — mampu menampilkan hasil pencarian wajah dari puluhan ribu foto dalam waktu kurang dari 2 detik.

My Dual Role

Tech Lead

Memimpin arsitektur sistem dan koordinasi tim pengembangan

Core Backend Engineer

Merancang dan membangun pipeline pengelolaan data foto serta sistem pencarian berbasis wajah dan bib number

The Challenge

Membangun platform yang dapat memproses dan mengindeks puluhan ribu foto event, lalu menyediakan pencarian berdasarkan wajah dan bib number dengan latensi di bawah 2 detik. Ini membutuhkan infrastruktur yang menggabungkan multiple technology stacks — masing-masing untuk tugas spesifik: facial recognition, API serving, data indexing, dan frontend delivery.

The Solution

Sebagai Core Backend Engineer, saya merancang pipeline pengelolaan data: Python untuk facial recognition processing, Golang untuk high-performance API indexing dan search, Laravel untuk business logic dan API gateway, PostgreSQL untuk data persistence, dan React JS di frontend. Arsitektur ini memungkinkan setiap komponen bekerja optimal pada tugasnya, menghasilkan pencarian wajah dari 50K+ foto dalam di bawah 2 detik.

Features

Key capabilities

Face Search

Cari foto event berdasarkan wajah. AI-powered facial recognition with sub-2s results.

Bib Number Search

Temukan foto berdasarkan nomor BIB peserta lomba atau event olahraga.

Bulk Photo Processing

Upload dan proses puluhan ribu foto event secara otomatis dengan indexing pipeline.

Event Management

EO dapat membuat event, mengelola album foto, dan memonitor aktivitas peserta.

Photo Downloads

Peserta dapat mengunduh foto berkualitas tinggi langsung dari hasil pencarian.

Admin Panel

Dashboard lengkap untuk manajemen event, foto, user, dan reporting analytics.
